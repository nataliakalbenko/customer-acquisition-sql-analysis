# Сustomer acquisition sql analysis- SQL Portfolio Project
## Project Overview
This project analyzes customer acquisition and customer purchasing behavior in an online store using SQL. The analysis is based on an educational dataset and focuses on identifying the most effective acquisition channels, payment services, and customer activity.

## Business Questions
The analysis answers the following questions:

- Through which acquisition channel were the largest number of **male customers** acquired?
- Which **payment service** was used for the highest number of orders?
- In which **city** was the highest number of orders placed?
- Who are the **Top 5 customers** who placed more than **10 orders**?

## Tools
- SQLite
- DBeaver

## Analysis Steps
- Imported CSV data into SQLite.
- Explored the database structure.
- Cleaned and prepared the data for analysis.
- Joined related tables using SQL JOINs.
- Applied filtering conditions (`WHERE`).
- Used aggregate functions (`COUNT`).
- Grouped data with `GROUP BY`.
- Filtered grouped results using `HAVING`.
- Sorted results using `ORDER BY`.
- Selected the Top 5 customers using `LIMIT`.

## Key Insights
- Instagram (Inst) was the acquisition channel that attracted the highest number of male customers.
- The most frequently used payment service was identified through order analysis.
- The city with the highest number of orders was determined.
- The Top 5 most active customers (with more than 10 orders) were identified.
